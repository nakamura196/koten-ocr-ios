# Changelog

## 1.3.0 (2026-03-22)

- NDLOCR-Lite（近代活字OCR）に対応。古典籍と近代の2つのOCRモードを切り替えて使用可能
- DEIMv2検出器とカスケードPARSeq認識器（30/50/100文字モデル）を追加
- 確認画面で「古典籍 OCR」「近代 OCR」をワンタップで選択
- OCR結果から「戻る」で確認画面に戻り、別モデルで再実行可能
- 認識処理の並列化（古典籍1.4x、近代6.7x高速化）
- スプラッシュ画面（アプリアイコン+タグライン）
- フィードバック送信機能（メール、デバイス情報自動付加）
- クラッシュリスクの修正（安全な配列アクセス、force unwrap除去）

## 1.2.2 (2026-03-19)

- App Storeマーケティング画像を更新（デバイスフレーム付き、JA/EN対応）
- スクリーンショット自動撮影・生成パイプラインを追加
- force unwrapの修正（ReadingOrder、CameraView）
- 空catchブロックにエラーログ追加
- ローカリゼーションを`String(localized:)`パターンに統一

## 1.2.1 (2026-03-14)

- チップ（Tip Jar）機能を追加

## 1.2.0 (2026-03-13)

- 現代語訳機能を追加（ローカルAI / クラウドAPI）
- トリミング確認画面を追加
- カメラ保存オプションを追加

## 1.1.0 (2026-03-10)

- カメラ権限フローの改善
- 設定画面にカメラ権限管理を追加
- ユニットテストを追加

## 1.0 (2026-03-04)

- 初回リリース
- NDL古典籍OCR-Liteモデルによるくずし字OCR
- カメラ撮影 / フォトライブラリからのOCR
- 認識結果のテキスト編集・エクスポート
- スキャン履歴
- ダーク/ライトテーマ
- 日本語/英語対応
